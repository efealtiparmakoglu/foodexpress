# 🍕 FoodExpress

Yemek sipariş platformu. Getir Yemek/Yemeksepeti tarzı.

## Özellikler
- 🏪 Restoran listeleme
- 🛒 Sepet yönetimi
- 📍 Konum bazlı arama
- 🔔 Canlı sipariş takibi
- 💳 Online ödeme
- ⭐ Değerlendirmeler
- 🏍️ Kurye takibi

## Teknolojiler
- React + React Native
- Node.js + Express
- MongoDB + Redis
- Google Maps API
- Stripe
